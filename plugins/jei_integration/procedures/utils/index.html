  • Does the recipe type [Recipe] have a result from the ingredients:
  • Get item in Slot [Slot] of block at x: [X] y: [Y] z: [Z]
  • Get fluid in Tank [Tank] of block at x: [X] y: [Y] z: [Z]

Used to get the amount of the output from the ingredients and the recipe type

Params:
- Output Type: the type of the output, either an item or a fluid
- Output Name: the name of the output
- Recipe: the recipe type
- Type: the type of the ingredient
- Name: the unique name set in the recipe type
- Consume: if true, it will consume the input if the recipe has an output
Return:
- the amount of the result from the recipe with the given ingredients

Used to check if a recipe with the given ingredients is valid

Params:
- Recipe: the recipe type
- Type: the type of the ingredient, either an item or a fluid
- Name: the unique name set in the recipe type
- Consume: if true, it will consume the input if the recipe has an output
Return:
- true, if the recipe with the given ingredients has an output
- false, if the recipe has no output

Gets the item from the slot of the block at a specific position

Params:
- Slot: the id of the slot
- X: the x coord of the block
- Y: the y coord of the block
- Z: the z coord of the block
Return:
- the itemstack in the slot, not a copy of the itemstack

Gets the fluid from the tank of the block at a specific position

Params:
- Tank: the id of the tank
- X: the x coord of the block
- Y: the y coord of the block
- Z: the z coord of the block
Return:
- the fluidstack in the tank, not a copy of the fluidstack

Sets the item in a slot of a block, while respecting the amount of the item in the slot

Params:
- Item: the itemstack that will be set in the slot
- Slot: the id of the slot
- X: the x coord of the block
- Y: the y coord of the block
- Z: the z coord of the block

Sets the fluid in a tank of a block, while respecting the amount of the fluid in the tank

Params:
- Fluid: the fluidstack that will be set in the tank
- Tank: the id of the tank
- X: the x coord of the block
- Y: the y coord of the block
- Z: the z coord of the block
