Result Lists
  • Get a list of [Output Type] output [Output Name] of the recipe type [Recipe] from the ingredients:
  • Is output list [List] empty
  • Get size of output list [List]
  • Get [Type] output from result list [List] at index [Index]
  • Does output list [List] contain [Type] output [Content]
  • Get random [Type] result from output list [List]
  • For each [Type] output from output list [List] as [Iterator] do
  • Add output list [List] to arraylist [Arraylist]

Used to get a list of every output from the given ingredients

Params:
- Output Type: the type of list
- Output Name: the name of the output
- Recipe: the recipe type
- Type: the type of the ingredient
- Name: the unique name set in the recipe type
- Consume: if true, it will consume the input once if the recipe has at least one output
Return:
- A list of every recipe output from the given ingredients

Used to get if the output list has any entries

Params:
- List: the output list
Return:
- true, if the list has entries
- false, if the list is empty

Gets the size of the output list

Params:
- List: the output list
Return:
- the size of the list

Gets the output at the specific index

Params:
- Type: the type of the output
- List: the output list
- Index: the index in the list
Return:
- the output at the specific index

Used to check if the specified content is in the output list

Params:
- List: the output list
- Type: the type of output
- Content: the content to check if it's in the list
Return:
- true, is the content is in the list
- false, if the list does not contain the item

Gets a random entry from the output list

Params:
- Type: the type of output
- List: the output list
Return:
- the random entry from the list

Iterates each entry in the output list as an output iterator

Params:
- Type: the type of output
- List: the output list
- Iterator: the output iterator block

An addon for the arraylist plugin allowing to add all entries from the output list to an arraylist

Params:
- List: the output list
- Arraylist: the arraylist the content is added to
