JEI Information

The information set here will be shown in the JEI information tab.

Information can either be set for items or for fluids.

Multiple items or fluids can be set to have information shown in the tab.

In the text list the information can be set. Each entry is a new line. To have an space between lines, an empty entry can be set in the text list.
