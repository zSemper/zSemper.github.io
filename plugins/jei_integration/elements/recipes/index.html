Custom Recipes

Custom recipes are the main way to set up recipes for the created recipe types.

The most important thing to select is the recipe type that the recipe uses.


To set an output slot the type of the output, either being an item, fluid, logic, number or text, and the name, that is set in the recipe type, needs to be set.

The same principle also applies when setting the inputs of the recipe. Any inputs that are marked as optional in the recipe type can be left out.
