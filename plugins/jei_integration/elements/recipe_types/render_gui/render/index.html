Category Rendering
  • Render text at x:[X] y:[Y] :[Text] with shadow:[Shadow]
  • Render tooltip at x:[X] y:[Y] and a width:[Width] height:[Height] with lines:[Lines]
  • Render texture [Name] with a texture x: [X] and texture y:[Y] at gui x:[GX] gui y:[GY] with width:[Width] height:[Height]
  • Get animated width from a max width:[Width] and a duration:[Duration]
  • Get animated height from a max height:[Height] and a duration:[Duration]

Used to get the result from the ingredients and the recipe type

Params:
- X: the x position of the text
- Y: the y position of the text
- Text: the text rendered in the JEI category
- Shadow: set true, if the rendered text should have a shadow

Used to get the amount of the result from the ingredients and the recipe type

Params:
- X: the x position of the tooltip field
- Y: the y position of the tooltip field
- Width: the width of the tooltip field
- Height: the height of the tooltip field
- Lines: the multiple lines shown as a tooltip

Renders a texture set in the Recipe Slot tab

Params:
- Name: the name of the set texture
- TX: the x position of the texture in the texture file
- TY: the y position of the texture in the texture file
- GX: the x position of the render in the gui
- GY: the y position of the render in the gui
- Width: the width of the texture in the texture file
- Height: the height of the texture in the texture file

Used to get an animated width that changes based on the duration

Params:
- Width: the width of the texture
- Duration: the time in ticks for the texture to be fully rendered
Return:
- a width based on the duration

Used to get an animated height that changes based on the duration

Params:
- Height: the width of the texture
- Duration: the time in ticks for the texture to be fully rendered
Return:
- a height based on the duration
