Used to get the dafault minecraft and any custom fluid in the workspace.

Params:
- Fluid List: a selection list for a fluid
Return:
- the fluidStack of the selected fluid

Convert the fluid at a block position into a fluidstack.

Params:
- X: the x coord of the block
- Y: the y coord of the block
- Z: the z coord of the block
Return:
- the fluid from the block position

An empty FluidStack, without a fluid and an amount.

Return:
- an empty fluidstack

Used to get a FluidStack with a specific amount.

Params:
- Fluid: the fluid used to get as an return
- Amount: the amount of the fluidstack
Return:
- the fluid with a specific amount

Sets the amount of the given fluidstack to a specific value.

Params:
- Fluid: the fluid to set the amount for
- Amount: the specific amount

Used to get the specific amount of the given fluidstack.

Params:
- Fluid: the fluidstack to get the amount for
Return:
- the amount of the fluidstack

Adds a specific amount to the given fluidstack.

Params:
- Amount: the amount that gets added to the fluidstack
- Fluid: the fluidstack to add the amount for

Removes a specific amount of the given fluidstack.

Params:
- Amount: the amount that gets removed from the fluidstack
- Fluid: the fluidstack to remove the amount from

Used to check if the given fluidstack is empty.

Params:
- Fluid: the fluidstack to check if it is empty
Return:
- true, if the fluidstack is empty
- false, if the fluidstack is not empty

Used to check if two fluidstacks are the same.

Params:
- Fluid1: the first fluid to compare to
- Fluid2: the second fluid to compare to
Return:
- true, if both fluidstacks are the same
- false, if both fluidstacks are different

Used to get the fluid from the registry name.

Params:
- Registry Name: the name of the fluid it is registered under
Return:
- the fluidstack the registry name responds to

Used to get the registry name from the fluidstack

Params:
- Fluid: the fluidstack to get the registry name
Return:
- the registry name of the fluidstack

Used to get the Display Name of the fluid.

Params:
- Fluid: the fluidstack to get the display name
Return:
- the display name of the fluidstack

Used to get the fluidstack from the given bucket item.

Params:
- Item: the bucket item to get the fluidstack from
Return:
- the fluidstack of the bucket

Used to get the bucket item of the given fluidstack.

Params:
- Fluid: the fluidstack to get the bucket item from
Return:
- the bucket item of the fluidstack
