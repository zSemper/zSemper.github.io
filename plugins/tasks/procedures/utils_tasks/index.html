Task Utilities
  • Does [Task Var] contain task [Task Name]

Can be used to check if a task variable has a task under a specific name.

Params:
- Task: the task variable
- Task Name: the name of the task to check
Return:
- true, if the task variable contains a task under the name
- false, if the task variable does not contain a task under the name
