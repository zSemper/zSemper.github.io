Procedures
  • Set Fluid [Fluid] with [Amount] mB in tank [Tank] of block at x:[X] y:[Y] z:[Z]
  • Get Fluid [Fluid]

Used to set the fluid in a tank of a block at specific position. This block will ONLY work correctly on mods created with this plugin, any other mod with multiple fluid tanks WON'T work!

Params:
- Fluid: the fluid the tank contents will be set to
- Amount: the amount of the fluid
- Tank: the index of the tank
- X: the x coord of the block
- Y: the y coord of the block
- Z: the z coord of the block

Used to select a fluid from the workspace. Work exactly like the Get Fluid block from fluid control.
